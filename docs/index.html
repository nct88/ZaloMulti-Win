v1.0.1 - Mới cập nhật

Quản lý nhiều nick Zalo Cực kỳ dễ dàng.

Sử dụng nhiều tài khoản Zalo độc lập trên Windows với dữ liệu biệt lập, bảo mật và hoàn toàn miễn phí. Giải pháp tối ưu cho làm việc và quản lý doanh nghiệp.

Tải về ngay Xem tính năng

Bản vá v1.0.1 có gì mới?

Chúng tôi vừa cập nhật những tính năng quan trọng nhất dựa trên phản hồi của bạn.

Cập nhật tự động

Hệ thống tự động kiểm tra phiên bản mới từ GitHub và thực hiện cập nhật silent chỉ với 1 click.

Sửa lỗi khởi chạy

Khắc phục hoàn toàn lỗi không có phản hồi khi nhấn nút "Mở tài khoản" trên một số phiên bản Windows.

Giao diện tối ưu

Sắp xếp lại các nút chức năng (Đưa nút Tạo lối tắt lên trên) giúp thao tác nhanh và thuận tiện hơn.

Tại sao chọn ZalỏMulti-Win?

Được xây dựng để đơn giản hóa quy trình làm việc của bạn.

Không giới hạn số nick

Mở bao nhiêu tài khoản tùy thích, chỉ phụ thuộc vào cấu hình RAM máy tính của bạn.

Cô lập dữ liệu tuyệt đối

Mỗi tài khoản có một môi trường APPDATA riêng, không bao giờ bị xung đột hoặc đá nick nhau.

Giao diện Acrylic/Glass

Hỗ trợ Dark Mode chuẩn Windows 11 và bảng màu sắc tùy chỉnh đa dạng cho từng icon.

Tạo Shortcut Desktop

Tự động tạo biểu tượng ngoài Desktop cho từng nick để truy cập trực tiếp không cần mở App quản lý.

Di chuyển Profile dễ dàng

Dễ dàng nén và chuyển toàn bộ dữ liệu tài khoản sang máy tính khác chỉ trong vài giây.

Mã nguồn mở 100%

Hoàn toàn miễn phí, minh bạch và an toàn. Bạn có thể kiểm tra mã nguồn bất cứ lúc nào.

Hướng dẫn cài đặt

Sẵn sàng sử dụng chỉ trong 4 bước đơn giản.

# Bước 1: Tải file từ GitHub (Nút GitHub phía trên)
# Bước 2: Giải nén vào thư mục bạn muốn lưu App
# Bước 3: Chuột phải vào file ZaloMulti.ps1 -> Run with PowerShell
# Bước 4: Thêm tài khoản mới và tận hưởng!
